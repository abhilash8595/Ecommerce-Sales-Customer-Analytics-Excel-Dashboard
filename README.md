# Ecommerce-Sales-Customer-Analytics-Excel-Dashboard
Ecommerce Sales & Customer Analytics Excel Dashboard

📊 Project Overview
This project presents an interactive Excel dashboard to analyze ecommerce sales and customer behavior. It provides insights into sales trends, customer demographics, order status, and channel performance to support data-driven decision-making.

🎯 Objectives
- Track overall sales performance and order trends  
- Analyze customer segments based on age and gender  
- Identify top-performing states and sales channels  
- Monitor order status like delivered, returned, cancelled  

🛠️ Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers (Category & Month filtering)  
- Data Cleaning & Transformation  

📁 Dataset Description
The dataset includes:
- Order ID  
- Sales Amount  
- State  
- Product Category  
- Customer Gender & Age Group  
- Order Status  
- Sales Channel (Amazon, Flipkart, Myntra, etc.)  
- Order Date (Month-wise)  

📊 Dashboard Features
- Sales: Men vs Women**  
  Women contribute 64% of total sales, while men contribute 36%  
- Top 5 States by Sales**  
  Maharashtra leads (~2.9M), followed by Karnataka and Uttar Pradesh  
- Orders vs Sales Trend**  
  Sales peak around March and gradually decline towards year-end  
- Order Status Breakdown**  
  ~92% orders successfully delivered  
  Very low return/cancellation rate (~3% each)  
- Sales Channels Analysis**  
  Amazon (35%) is the dominant platform  
  Flipkart (22%) and Myntra (23%) also contribute significantly  
- Age vs Gender Insights**  
  Adult women segment contributes the highest (~34.59%)  
  Teenagers and seniors have comparatively lower contribution  

📈 Key Insights
- Female customers are the primary revenue drivers  
- Maharashtra is the highest revenue-generating state  
- Amazon is the most effective sales channel  
- Business has strong delivery success rate (92%)  
- Sales show slight seasonal variation with peak in Q1  

🚀 How to Use
1. Download the Excel file  
2. Open in Microsoft Excel  
3. Use slicers (Category & Month) to filter data  
4. Interact with charts for insights
